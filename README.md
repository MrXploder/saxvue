# Saxvue Next (Vue 3)

This package is a Vue 3 component library (Saxvue fork) intended to be consumed by other projects. Storybook is included to showcase every component and its key options.

## Storybook

- `npm install`
- `npm run storybook`

Build a static Storybook with `npm run build-storybook`.

## Structure

- `src/components` – core Saxvue components
- `src/layout` – grid and layout primitives
- `src/stories` – Storybook stories for every component
