# Data Display Components

Components for presenting data: avatars, cards, tables, and pagination.

---

## SvAvatar

A user avatar with badge, loading, and writing indicator support.

### Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `string \| number` | `""` | Avatar size (CSS value or number) |
| `circle` | `boolean` | `false` | Circular shape |
| `square` | `boolean` | `false` | Square shape |
| `history` | `boolean` | `false` | Story-style ring |
| `historyGradient` | `boolean` | `false` | Gradient story ring |
| `loading` | `boolean` | `false` | Loading overlay |
| `writing` | `boolean` | `false` | Typing indicator dots |
| `badge` | `boolean` | `false` | Show badge dot |
| `badgeColor` | `string` | `""` | Badge dot color |
| `badgePosition` | `string` | `null` | Badge position |
| `pointer` | `boolean` | `false` | Cursor pointer on hover |

### Slots

| Slot | Description |
| --- | --- |
| `default` | Avatar image (typically an `<img>`) |
| `text` | Text initials when no image |
| `badge` | Custom badge content |

### Usage

```html
<sv-avatar>
  <img src="/avatar.jpg" alt="User" />
</sv-avatar>

<sv-avatar circle history-gradient>
  <template #text>JD</template>
</sv-avatar>

<sv-avatar badge badge-color="danger" size="60">
  <img src="/avatar.jpg" alt="User" />
  <template #badge>3</template>
</sv-avatar>
```

### SvAvatarGroup

Stack avatars with overlap:

```html
<sv-avatar-group max="3">
  <sv-avatar><img src="/a.jpg" /></sv-avatar>
  <sv-avatar><img src="/b.jpg" /></sv-avatar>
  <sv-avatar><img src="/c.jpg" /></sv-avatar>
  <sv-avatar><img src="/d.jpg" /></sv-avatar>
</sv-avatar-group>
```

## SvCard

A flexible card container with image, title, text, buttons, and interaction slots.

### Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `type` | `string \| number` | `"1"` | Card type/layout variant |

### Slots

| Slot | Description |
| --- | --- |
| `img` | Card image/media area |
| `title` | Card title |
| `text` | Card body text |
| `buttons` | Action buttons row |
| `interactions` | Overlay interactions on the image |

### Usage

```html
<sv-card>
  <template #title><h3>Card Title</h3></template>
  <template #text><p>Card body content goes here.</p></template>
  <template #buttons>
    <sv-button flat>Read More</sv-button>
  </template>
</sv-card>
```

### SvCardGroup

Grid/slider of cards:

```html
<sv-card-group>
  <sv-card>…</sv-card>
  <sv-card>…</sv-card>
  <sv-card>…</sv-card>
</sv-card-group>
```

## SvTable

A data table with automatic column detection, striped rows, and sort capabilities.

### Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `striped` | `boolean` | `false` | Alternating row colors |

### Sub-components

- **`<vs-tr>`** — Table row, accepts `data` prop and `isSelected` boolean
- **`<vs-th>`** — Table header cell, accepts `sort` boolean
- **`<vs-td>`** — Table data cell

### Slots

| Slot | Description |
| --- | --- |
| `header` | Header area above the table |
| `thead` | Custom `<thead>` content |
| `tbody` | Custom `<tbody>` content |
| `notFound` | Shown when no data matches |

### Usage

```html
<sv-table striped>
  <template #thead>
    <vs-tr>
      <vs-th sort>Name</vs-th>
      <vs-th sort>Email</vs-th>
      <vs-th>Actions</vs-th>
    </vs-tr>
  </template>
  <template #tbody>
    <vs-tr v-for="user in users" :key="user.id" :data="user">
      <vs-td>{{ user.name }}</vs-td>
      <vs-td>{{ user.email }}</vs-td>
      <vs-td><sv-button flat size="small">Edit</sv-button></vs-td>
    </vs-tr>
  </template>
</sv-table>
```

## SvPagination

Page navigation with arrow buttons, dotted ranges, and progress indicator.

### Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `number` | — | Current page (`v-model`) |
| `length` | `number` | `0` | Total number of pages |
| `max` | `number` | `9` | Maximum visible page buttons |
| `infinite` | `boolean` | `false` | Infinite loop mode |
| `progress` | `boolean` | `false` | Show progress bar |
| `notMargin` | `boolean` | `false` | Remove spacing between buttons |
| `buttonsDotted` | `boolean` | `false` | Dotted button style |
| `notArrows` | `boolean` | `false` | Hide prev/next arrows |
| `onlyArrows` | `boolean` | `false` | Show only arrows (no page numbers) |
| `circle` | `boolean` | `false` | Circular buttons |
| `square` | `boolean` | `false` | Square buttons |
| `disabled` | `boolean` | `false` | Disabled state |
| `disabledItems` | `number[]` | `[]` | Specific page numbers to disable |
| `loadingItems` | `number[]` | `[]` | Specific page numbers with loading state |
| `dottedNumber` | `number` | `5` | Number of dots in collapsed range |

### Slots

| Slot | Description |
| --- | --- |
| `default` | Custom page button content (receives page number) |

### Usage

```html
<sv-pagination v-model="page" :length="20" />

<sv-pagination v-model="page" :length="50" progress circle />

<sv-pagination v-model="page" :length="10" :disabled-items="[3, 7]" />
```
